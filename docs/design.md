# Design - v2


```shell
bm --save <link>
bm --save <link> <category...>  # save the link and associate it with given categories
bm --list # list all of the bookmarked links
bm --list <category> # list all of the bookmarked links in given category
bm --delete <ith bookmark> # delete the ith bookmarked
bm --delete <category> # delete the given category
bm --update i <new link>  # update the ith bookmark with new link
bm --open i # open given link id in default browser
bm --export <directory> # exports links to given directory
bm --help # prints usage
bm --version  # prints version 
```
